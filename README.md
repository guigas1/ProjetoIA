O sistema permite:

Recomendar músicas para novos usuários (cold start) com base em preferências
Encontrar músicas similares a partir de uma música escolhida
Visualizar resultados em uma interface interativa desenvolvida com Streamlit

Como Executar o Projeto
1. Clonar ou baixar o projeto
git clone <url-do-repositorio>
cd <nome-da-pasta>

2- python3 -m venv venv
source venv/bin/activate

3- pip install streamlit pandas numpy scikit-learn

4- streamlit run projetoIA.py
